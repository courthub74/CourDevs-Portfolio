About Me:

My name is Courtney Jacobs. I have had a fascination with coding since my first internship, studying computer science in college. I was a support technician for a Software as a service company. What started as a mere one day a week gig turned into a 7 year long work experience, which later elevated to team supervisor as the company grew. Part of my responsibility was to test and debug software. I gained much experience with the SDLC (Software Development Life-Cycle) and fell in love with the process of building an application with syntax ever since. Web Application development is something I would pursue as a hobby if money was no object.

I am a solutions inclined, analytical, as well as creative Full Stack Web Developer with experience in design, development, testing, and implementation of web application software.



Areas of focus include these languages, technologies and frameworks:

| Python | Django | React | JavaScript | API Integration and Development | JSON | XML | PostGreSQL | Git | GitHub | JQuery |
